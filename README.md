# OnGameShorts

Windowsの画面の最上部に常駐する、縦長ウィンドウ型のYouTube Shortsプレイヤーです。
ゲーム・作業アプリなど、他のアプリを操作している最中でも、フォーカスを奪わずに常駐表示できます
(Rusty's Retirementのような「画面下部の帯ウィンドウ」形式)。

Googleアカウントにログインした状態のYouTube(スマートフォン版)を埋め込み表示し、YouTube自身の
「おすすめ」でShortsを連続再生します。動画の選定・キュー管理はこちらでは行わず、YouTubeの推薦に任せる方式です。
埋め込みブラウザはスマートフォンとして振る舞う(User-Agentを偽装する)ことで、YouTube側に
帯ウィンドウの縦長で小さいサイズにぴったり収まるモバイル向けレイアウトを出させています。

キーボード操作はグローバルホットキー(`RegisterHotKey`)のみで行い、低レベルのキーボードフックは使用しません。
他アプリのプロセスへの介入(メモリ読み取り・DLL注入・オーバーレイ注入など)は一切行わない、独立したウィンドウアプリです。

## ダウンロードしてすぐ使う

1. [Releases](../../releases) ページから最新の `OnGameShorts-win64.zip` をダウンロードします。
2. ダウンロードしたzipを右クリック→「すべて展開」で好きな場所に展開します(デスクトップなどでOK)。
3. 展開してできた `OnGameShorts` フォルダの中の `OnGameShorts.exe` をダブルクリックで起動します。
   - 初回はWindowsの「発行元不明」の警告が出ることがあります。「詳細情報」→「実行」で起動できます。
4. 画面下部に帯ウィンドウが表示されます。タスクトレイ(画面右下の通知領域)のアイコンを右クリック→
   「Googleアカウントにログイン...」から一度だけログインしてください。
5. 終了したいときは、タスクトレイアイコンを右クリック→「終了」を選びます。

インストーラーではないので、フォルダごと削除すればアンインストール完了です
(設定やログイン情報は `%APPDATA%\OnGameShorts` に保存されるので、そちらも消せば完全に削除できます)。

## 開発者向け: ソースから実行する

Windows専用です。既存のPython環境を汚さないよう、必ず仮想環境を使ってください。

```powershell
py -3.11 -m venv .venv
.venv\Scripts\pip install -r requirements.txt
```

### 起動

```powershell
.venv\Scripts\python main.py
```

タイトルバーの無いウィンドウなので、終了はタスクトレイアイコンの右クリックメニュー「終了」から行います。

### exe(配布用パッケージ)を作る

```powershell
.venv\Scripts\pip install pyinstaller
.venv\Scripts\pyinstaller --name OnGameShorts --windowed --noconfirm main.py
```

`dist\OnGameShorts\` フォルダ一式(`OnGameShorts.exe` を含む)が生成されます。このフォルダをそのままzip化して配布します。

## ログイン(初回のみ)

タスクトレイアイコンを右クリック→「Googleアカウントにログイン...」で、通常サイズの
ログイン専用ウィンドウが開きます。帯ウィンドウ自体は小さくフォーカスも奪わない作りなので、
ログイン入力はこちらの専用ウィンドウで行ってください。

ログイン情報(Cookie等)はプロジェクト直下の `profile_data/` に保存され、帯ウィンドウの
埋め込みブラウザと共有されます。ログイン後、タスクトレイの「再読み込み」を選ぶと帯ウィンドウ側にも反映されます。

初回アクセス時にYouTubeのCookie同意バナーが出ることがありますが、ログインウィンドウで一度
選択すれば以降は表示されなくなります。

## 使い方

画面下部・タスクバー直上に帯ウィンドウが表示され、中央に縦動画プレイヤー、左に再生中タイトル、右にホットキー一覧が並びます。
ウィンドウはフォーカスを奪わないので、他のアプリを操作したままホットキーで操作できます。

動画IDなどの登録は不要です。ログインさえしていれば、起動時に自動でYouTubeのホームから最初のShortsへ移動し、
以降はYouTube自身の「おすすめ」に沿って次のShortsへ流れていきます。

帯ウィンドウの左右パネル部分(タイトル・ホットキー一覧のあたり)はドラッグして好きな位置に移動できます。
右上の✕ボタンでも非表示にできます。

| ホットキー (既定) | 動作                                          |
| ----------------- | --------------------------------------------- |
| Ctrl+Alt+P        | 再生 / 一時停止 トグル                        |
| Ctrl+Alt+N        | 次のShortsへスキップ                          |
| Ctrl+Alt+H        | ウィンドウの表示/非表示 トグル(✕ボタンと同じ) |
| Ctrl+Alt+S        | 設定画面を開く                                |

すべて `config.json` の `hotkeys` で変更できます。使用中の他アプリのキー割り当てと被らないものを選んでください。

## 設定画面

タスクトレイアイコン→「設定...」で、ダークなカード型の設定画面が開きます。`config.json`を直接編集しなくても、
以下をアプリ内から変更できます。

- **使い方**: 基本操作とホットキー一覧の説明(初回起動時は自動でこの画面が開きます)
- **ウィンドウ**: 幅/高さ/画面下端からの余白、自動配置のON/OFF・手動座標
- **ホットキー**: 各操作のキー割り当て(重複していると保存時に警告が出ます)
- **表示**: `compact_style`のON/OFF

「保存」を押すとウィンドウサイズ・位置とホットキーはその場で反映されます(表示設定はタスクトレイの「再読み込み」または次回起動で反映)。

## 既知の制限

- 実際のyoutube.comページ(モバイル版)を埋め込んでいるため、再生/一時停止・次へのスキップはページに向けて
  キー操作をシミュレートすることで実現しています。YouTube側のUI/DOM変更で動かなくなる可能性があります。
- 起動時は`https://www.youtube.com/shorts`(動画IDなし)を開いており、ログイン済みであれば
  YouTube側がおすすめのShortsへ自動でリダイレクトします。ごく稀にリダイレクトされない場合は、
  タスクトレイの「再読み込み」を試すか、ウィンドウ内で手動でShortsを1本開いてください。
- モバイル版レイアウトのおかげで動画は帯ウィンドウのサイズにそのまま収まりますが、上部のツールバーなど
  一部の装飾は`compact_style`(既定on)で非表示にしています。YouTube側の変更に弱い実装のため、
  不具合が出た場合は`config.json`の`compact_style`を`false`にして素の表示に戻せます。

## config.json 主要項目

| キー                      | 説明                                                   |
| ------------------------- | ------------------------------------------------------ |
| `compact_style`           | YouTubeページのヘッダー/コメント欄などを非表示にするか |
| `window.width` / `height` | 帯ウィンドウのサイズ (既定 600x340)                    |
| `window.x` / `y`          | ウィンドウ位置。`null`なら画面下部中央に自動配置       |
| `window.margin_bottom`    | 画面下端(タスクバー上端)からの余白                     |
| `hotkeys.*`               | 各操作に割り当てるホットキー (`modifiers` + `key`)     |

## 構成

```
main.py                起動処理・全体の配線・システムトレイ
config.json             設定ファイル
app/config.py           設定の読み込み/保存
app/band_window.py       帯ウィンドウ本体、YouTube埋め込み、ホットキー登録
app/login_window.py      ログイン専用の通常ウィンドウ
app/browser_profile.py   帯ウィンドウ/ログインウィンドウ共有の永続ブラウザプロファイル
app/hotkeys.py           RegisterHotKeyラッパー
app/settings_dialog.py   設定画面(使い方/ウィンドウ/ホットキー/表示)
profile_data/            ログインCookie等(gitignore対象)
```
